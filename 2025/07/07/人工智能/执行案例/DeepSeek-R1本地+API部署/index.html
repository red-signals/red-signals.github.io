DeepSeek-R1本地+API部署

视频来源：安格视界 - DeepSeek‑R1 本地+API最强部署

本笔记整理自安格视界频道的演示视频，涵盖如何在本地部署 DeepSeek‑R1 模型架构（支持 8b / 70b / 671b），并通过 API 接入实现联网搜索、公网访问与全平台终端兼容，适合 AI 爱好者、开发者、企业实践者参考。

✨ 功能亮点

  • ✅ 支持多种模型体积：8b、70b、671b
  • 🌐 联网搜索：本地部署也能访问互联网内容
  • 🔗 公网访问：支持外部设备调用本地服务
  • 🧠 强大推理：复杂逻辑、多轮问答无压力
  • 💻 全终端可用：PC、移动端统一接口体验
  • 🛠️ 搭配工具：Ollama + OpenWebUI 构建完整体验

🧱 部署流程概要

步骤 操作内容
1️⃣ 环境准备 安装依赖，准备 GPU 资源（建议 24G 以上显存）
2️⃣ 模型下载 按需获取 DeepSeek‑R1 的 8b / 70b / 671b 模型
3️⃣ 启动本地服务 使用 Ollama 进行模型载入与 API 暴露
4️⃣ 接入界面 使用 OpenWebUI 构建图形化交互
5️⃣ 联网功能 启用插件，设置联网检索能力
6️⃣ 公网访问 配置端口映射 / Cloudflare Tunnel 等方式对外暴露
7️⃣ 终端联调 确保多设备、移动端访问无障碍

📍 技术细节与关键工具

  • Ollama：本地模型运行平台，支持 DeepSeek‑R1 等大模型部署。

  • OpenWebUI：图形化界面，提供聊天、联网搜索、文件上传等功能。

  • 模型兼容性：

    • 8b：轻量级部署、适配中端显卡；
    • 70b：强推理能力，建议多卡或高显存；
    • 671b：需 A100 / H100 等数据中心级硬件。

  • 联网功能：

    • 配置联网插件或调用如 browser、web-research 插件；
    • 可实现实时 web 检索 + 本地推理结合。

💡 实用建议

  • 模型部署建议使用 Linux + Docker + GPU 驱动完整 的环境；
  • 公网访问需特别注意端口防火墙配置与身份认证；
  • 联网插件可能需要科学网络支持（建议代理配置）；
  • 多模型并存部署建议使用不同端口 + Nginx 做反代；
  • 如果资源有限，建议优先部署 8b 版本体验功能全貌。

📦 应用场景推荐

  • 🤖 本地私人助手：完全离线也能进行强逻辑对话；
  • 🏢 企业内网知识库：通过接口接入自有数据或系统；
  • 🌍 混合智能搜索：结合联网搜索+本地推理的混合 AI 体验；
  • 📱 跨平台助理：同一个模型服务同时支持桌面和移动客户端访问。

⚠️ 注意事项

  • 高性能模型（如 70b、671b）需高端硬件支持；
  • API 对公网暴露需配置访问控制与认证机制；
  • 模型部署与网络功能需合规使用，遵守版权与数据法规；
  • 公网服务建议配合 Cloudflare Zero Trust、frp、ngrok 等工具。

🔗 资源与工具整合

本节汇总与 DeepSeek‑R1 本地部署、API 接入相关的重要工具、API 平台与环境配置，便于开发者快速查阅与接入使用。

🧱 Ollama 本地部署工具

  • 官方网址：https://ollama.com
  • 用于本地运行大模型（如 DeepSeek-R1），配合 OpenWebUI 使用效果更佳。

✅ 推荐环境变量配置：

OLLAMA_MODELS=[模型地址]       # 指定模型存储路径或远程地址
OLLAMA_HOST=0.0.0.0           # 监听所有网络接口
OLLAMA_ORIGINS=*             # 允许所有来源访问（如 WebUI）

🌐 Open-WebUI 前端交互界面

  • GitHub 仓库：open-webui/open-webui
  • 搭配 Ollama 实现图形化对话界面。

🚀 Docker 快速部署命令：

docker run -d -p 8082:8080
 -v open-webui:/app/backend/data
 –name Open-WebUI
 –restart always
ghcr.io/open-webui/open-webui:main

🔌 API 平台汇总

以下为可直接调用 DeepSeek‑R1 的云端 API 服务，支持推理与联网搜索功能。

1️⃣ 硅基流动 API

  • 官网：https://cloud.siliconflow.cn/i/jPI17VGX
  • 接口地址：https://api.siliconflow.cn/v1
  • 模型名称：deepseek-ai/DeepSeek-R1

2️⃣ Groq API

  • 官网：https://groq.com
  • 接口地址：https://api.groq.com/openai/v1
  • 模型名称：deepseek-r1-distill-llama-70b

3️⃣ 芝麻开门 AI 接口

  • 官网：https://ai.opendoor.cn

  • 接口地址：https://ai.opendoor.cn

  • 支持模型：

    • DeepSeek-R1：标准聊天模型
    • DeepSeek-R1-all：集成联网搜索版本

🔄 DeepSeek API 显示思考过程转换工具

  • 项目名称：DeepSeek2ChatWithThought
  • GitHub 地址：https://github.com/chadyi/DeepSeek2Ch..
  • 功能：将 DeepSeek 接口调用包装为可视化显示思考过程的对话流，有助于调试与交互优化。
